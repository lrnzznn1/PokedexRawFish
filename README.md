# PokedexRawFish

PokedexRawFish is an Android application that allows users to browse a list of Pokémon fetched from the PokeAPI and view detailed information about each Pokémon. 
It uses Kotlin, Kotlin Coroutines, Jetpack Compose, Room Database, Kotlin Serialization, Ktor HTTP client, and other modern Android development libraries.

## Features

- Fetches a list of Pokémon from the PokeAPI.
- Displays Pokémon list with pagination using Jetpack Compose.
- Allows users to view detailed information about each Pokémon, including their moves and sprites.
- Persists Pokémon data locally using Room Database with Kotlin Serialization for complex data types.
- Handles network requests asynchronously using Kotlin Coroutines and Ktor HTTP client.
- Checks internet connectivity before making network requests.

## Requirements

- Android Studio Arctic Fox (2020.3.1) or later
- Android SDK Version 21 (Lollipop) or higher
- Kotlin Version 1.5 or higher
- Kotlin Coroutines
- Jetpack Compose Version 1.0 or higher
- Room Database
- Ktor HTTP Client
- Kotlin Serialization

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/lrnzznn1/PokedexRawFish
   ```
2. Open the project in Android Studio.

3. Build and run the project on an Android emulator or device.
